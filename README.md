1D Array Practice in C

This repository contains my practice programs while learning 1D Arrays in C Programming.
The goal of this practice set is to build a strong foundation in array manipulation, looping, and problem-solving before moving to more advanced topics like 2D Arrays, Sorting, and Data Structures.

📚 Topics Covered

Array declaration and initialization

Taking array input from users

Traversing arrays using loops

Basic array operations

Searching elements in arrays

Logical problem solving using arrays

🧠 Practice Problems

The following problems were solved as part of my learning process:

Print all elements of an array

Take user input and store elements in an array

Find the sum of all array elements

Calculate the average of array elements

Find the largest element in an array

Find the smallest element in an array

Count even numbers in an array

Count odd numbers in an array

Find sum of even and odd elements separately

Search for an element in the array

Reverse the array

Print elements at even indices

Print elements at odd indices

Find the second largest element

Check whether an array is a palindrome

🛠️ Language Used

C Programming

🎯 Learning Outcome

After completing these problems, I gained a clear understanding of:

How arrays work internally

Index-based data access

Looping through arrays efficiently

Solving logical problems using arrays

2D Arrays
